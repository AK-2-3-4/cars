🚗 AI Interactive Car Showroom (Unity)
Overview

This project is an interactive 3D car showroom built using Unity, where users can explore different cars, apply AI-inspired styling themes, and view vehicles from multiple cinematic camera angles.

The goal of this project is to demonstrate how generative AI concepts can enhance product visualization and interactive design environments.
✨ Features
🚘 Car Selection

Users can switch between multiple car models in the showroom.
Each car is displayed in an interactive 3D environment.

🎨 AI-Based Styling

Users can enter a theme (e.g., sporty, luxury, military).
The system dynamically applies styling changes such as:
Body color
Material appearance
Tire styling

🎥 Multiple Camera Views

Users can inspect the car from different angles:

Front view
Side view
Rear view
Top view
Orbit camera mode

🔧 Tyre Customization
The system dynamically detects tyre or wheel materials and applies theme-based styling.

🖥 Interactive UI

Next Car
Apply Style
Change Tyres
Camera view navigation

🛠 Technologies Used

Unity Engine
C#
TextMeshPro UI
Basic AI-style theme generation logic
3D assets and materials

🧠 AI Integration
The project includes AI-inspired styling logic where users input a theme, and the system generates visual styling suggestions for the car.
This demonstrates how generative AI ideas can be integrated into interactive 3D applications.

⚙️ Project Structure
Assets/
Packages/
ProjectSettings/

These folders contain all Unity scripts, UI elements, car models, and project configuration.

🎮 How to Run
Open the project in Unity Hub
Load the main scene
Click Play

Interact with the UI to:
Switch cars
Enter styling themes
Change tyre styles
Navigate camera views
